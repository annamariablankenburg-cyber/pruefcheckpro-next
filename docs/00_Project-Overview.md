# PrüfCheckPro – Project Overview

## Kurzbeschreibung

PrüfCheckPro ist eine digitale Plattform für Baustoffprüfung.

Die Plattform verbindet Lernen, Prüfverfahren, Probekörperverwaltung, Rechner, Kalender, Exportfunktionen und PrüfCheck AI in einer modernen Web-App.

## Claim

Baustoffprüfung digital.

- Apple
- Linear
- Stripe

Die Oberfläche soll hochwertig, ruhig, modern und professionell wirken.

## Projektprinzipien

1. Einfach statt kompliziert
2. Praxis vor Theorie
3. Eine Plattform statt viele Einzellösungen
4. Professionelles Design
5. KI unterstützt, ersetzt aber kein Fachwissen
