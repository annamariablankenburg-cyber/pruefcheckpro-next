# Entwicklungsprinzip

Jede neue Funktion muss:

- mobil funktionieren
- schnell sein
- wiederverwendbar sein
- dokumentiert werden
- getestet werden

---

# Release-Ziel

Version 1.0 soll von Azubis im Alltag genutzt werden können.

Version 2.0 soll in Baustofflaboren einsetzbar sein.

Version 3.0 richtet sich an Unternehmen mit mehreren Standorten.
